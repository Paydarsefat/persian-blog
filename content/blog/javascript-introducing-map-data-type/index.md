یکی از ویژگی‌های جالب ES6 کار با Data Type ای به نام Map هست که در اصل نمونه‌ی ساده شده‌ای از کار با Array ها در جاوا اسکریپت هست.

<br />

```javascript
let m = new Map()

m.set('name', 'Ehsan')
m.set('surname', 'Gazar')

// ---
m.get('name')
```
